# Images du site et illustrations

Ce dossier contient les images utilisées pour le site et ses illustrations (fonds de section, visuels, etc.).

Vous pouvez aussi héberger ces images dans le bucket Supabase **site-images** (créé par la migration `139_storage_site_images_bucket.sql`) et mettre à jour les URLs dans le code si besoin.

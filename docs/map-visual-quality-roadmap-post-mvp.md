# Roadmap visuelle post-MVP perf

## Regle

Chaque amelioration visuelle doit etre derriere:

- un flag
- un budget de perf mesure
- un rollback simple

## Backlog priorise

1. Rehausse legere des labels (densite progressive).
2. Styles hydro enrichis (rivieres/lacs) sans animation lourde.
3. Effets de surbrillance localises sur selection.
4. Theme "riche" optionnel pour devices capables.

## Conditions d'activation

- SLO perf deja respectes en palier `perf`.
- Tests mobile valides.
- Pas de regression CI benchmark.
